# Changelog

所有重要功能变更记录在此文件中。

## [Unreleased]

### Added
- 面板顶部左侧新增模式切换按钮：`App`（应用面板）与`终端`（占位页）。
- Launchpad 新增“一键智能合并”，支持按用途、开发者、首字母分组。
- 智能合并增强 macOS 系统自带应用聚类，优先归入「系统实用工具」与「macOS 自带应用」文件夹。
- 新增「全屏时隐藏 pill」开关（仅收缩态生效），并限制为浏览器/播放器场景触发。
- 动画设置改为卡片式选项，且每类仅保留三组动画（弹性/快速/柔和）。

### Changed
- 设置页预览中的 pill 与展开面板样式对齐真实面板形态。
- 设置侧边栏移除「通用」页签，退出按钮下沉到侧边栏底部。
- 辅助功能权限入口改为与其它开关一致的交互样式。

### Fixed
- 修复收起动画被展开动画覆盖导致“收起动画不生效”的问题。
- 修复 Launchpad 首次加载偶发空扫描导致布局被误清空的问题（空 app 列表不再触发布局清理）。
- 修复单侧信息位时空侧仍出现黑色占位的问题。
- 去除面板外阴影与顶部分隔线，降低“灰边/边框感”，统一纯黑视觉。
